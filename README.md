# Типографинатор

Небольшой инструмент для автоматического форматирования текста, который расставляет неразрывные пробелы и дефисы там, где это необходимо для правильных переносов.

Создан под задачи вёрстки больших текстов, где типографика и контроль за переносами имеют значение.

Попробовать инструмент [вживую](dtsiki.github.io/typographinator/)

## О проекте

Инструмент написан на базе [Create React App](https://github.com/facebook/create-react-app) и двух чашек кофе.

Чтобы запустить проект локально:

1. Склонируйте репозиторий:

```bash
git clone https://github.com/dtsiki/typographinator.git
```

2. Перейдите в директорию проекта:

```bash
cd typographinator
```

3. Установите зависимости и запустите сервер:

```bash
npm install
npm start
```

4. Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000).
